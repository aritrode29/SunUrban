# Digital Twin Licensing Revenue Stream

## Overview

**Parking-Lot Digital Twin Licensing** - Sell access to the data layer from solar canopy installations. This creates a scalable SaaS revenue stream on top of the hardware infrastructure.

---

## What Is It?

A **data licensing product** that monetizes the rich data generated by solar canopy installations. Each canopy site becomes a data collection point, creating valuable datasets for multiple customer segments.

## Revenue Model

### Pricing Structure

**Per-Site Licensing**: $25k/year per site
- Includes access to all data types
- API access
- Dashboard access
- Historical data (1 year)

**Per-Data-Type Licensing**: $5-10k/year per data type
- Customers can select specific data types
- Lower cost for focused needs

**Enterprise Licensing**: $50-100k/year
- Multiple sites
- Custom data packages
- Priority support
- Extended historical data

### Revenue Projection

**For 3 Sites (Current):**
- Base: $25k/site × 3 = **$75k/year**
- With multiple customers per site: **$75-150k/year**

**For 10 Sites (Scale):**
- Base: $25k/site × 10 = **$250k/year**
- With multiple customers: **$250-500k/year**

**For 50 Sites (Full Scale):**
- Base: $25k/site × 50 = **$1.25M/year**
- With multiple customers: **$1.25-2.5M/year**
